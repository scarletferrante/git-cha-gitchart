# git-cha-gitchart
This program will git you a gitchart 🛁 

This thing will output a chart of all of your commits over time. 

## Installation
`pip install git-cha-gitchart`

## Usage
1. Navigate to your git repo
2. Run this `git-cha-gitchart`
